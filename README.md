# My-Des
Symmetric Key Encryption ~~ Des

# 语言 & 环境
（1）语言
     Python
（2）环境
     Python版本：python 3.7
     Python工具：PyCharm Professional
     
# 操作流程
（1）在 < text.txt > 文件中读取需要加密的文本
（2）在 < key.txt > 文件中写入随机产生固定长度的Key
（3）在 < ciphertext.txt > 文件中写入DES加密、Base64编码后的数据
（4）在 < plaintext.txt > 文件中写入Base64解码、DES解密后的数据
